关于macOS深色模式texstudio显示问题的解决方案

blackdogtop
2020-05-16
macOS

环境

macOS 10.15.4
texstudio 2.12.22

问题说明

macOS深色模式会导致texstudio文本可读性变差（见页面左半部分文本）

解决方案

1. 更改texstudio设置(不推荐)

更改 texstudio 中 Preferences - Syntax Highlighting - Line highlighting - current
将其中 current 的 background color 修改成相对可见的颜色。
此方法就是修改选中的文本行的背景颜色，但会导致 texstudio 在 macOS 切换为浅色模式下显示效果不佳，故不推荐。

defaults
在 macOS X 中 defaults 命令被用于查看和修改设置。大多应用程序偏好设置中的选项被存储在以 .plist 为后缀的文件中，但这其中不乏有一些隐藏设置是在偏好设置中不可见，即可通过 defaults 命令更改。

defaults 搭配 read 用于查看内容，搭配 write 用于写入/修改内容，搭配 delete 用于删除内容。

NSRequiresAquaSystemAppearance
对于 NSRequiresAquaSystemAppearance 及 为macOS应用选择特定外观，苹果提供了官方的开发者文档，可参考 Apple 开发者

NSRequiresAquaSystemAppearance 被用于选择退出深色模式。可以通过修改 Info.plist 文件中 NSRequiresAquaSystemAppearance 的值为 true 来退出深色模式。
